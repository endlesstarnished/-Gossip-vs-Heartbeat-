# Задание 2: Анализ протоколов обнаружения отказов (Gossip vs Heartbeat)

## Файлы в репозитории
- `bandwidth_calculator.py` - Python функции для расчета пропускной способности
- `Расчет.txt` - формулы и расчеты для 50 и 100 узлов
- `сравнительная таблица.png` - сравнение характеристик протоколов
- `recommendations.txt` - рекомендации по выбору протокола
- `результат эксперимента.png` - скриншот эксперимента с Serf (статус failed)
- `output.png` / `output.txt` - вывод программы bandwidth_calculator.py

## Результаты
- Gossip для 50 узлов: 0.6 Мбит/с
- Heartbeat для 50 узлов: 0.2 Мбит/с
- Экспериментальное время конвергенции Gossip: ~5 секунд (для 3 узлов)
